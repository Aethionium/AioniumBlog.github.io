# Aionium Blog

Personal blog website built with HTML, CSS, and JavaScript.

## GitHub Pages Setup

This site is deployed on GitHub Pages.

### Access URL
- If repository name is `aethionium.github.io`: https://aethionium.github.io/
- If repository name is something else (e.g., `blog`): https://aethionium.github.io/blog/

## Local Development

Simply open `index.html` in a browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server
```

Then visit: http://localhost:8000
